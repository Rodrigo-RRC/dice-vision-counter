# dice-vision-counter

Computer vision system that detects dice in images and sums their face values, built with Python and OpenCV.

## 📋 Sobre

Sistema que recebe uma imagem contendo múltiplos dados em um fundo neutro
e calcula a soma das faces visíveis, utilizando técnicas de visão
computacional (segmentação, detecção de contornos/blobs).

## 🛠️ Tecnologias

- Python 3.x
- OpenCV
- NumPy
